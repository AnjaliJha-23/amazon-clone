# Amazon Clone

A front-end clone of the Amazon homepage built using HTML and CSS.

## Features

- Responsive navigation bar
- Hero section
- Product cards
- Footer
- Amazon-inspired layout
- Clean UI using Flexbox

## Technologies Used

- HTML5
- CSS3
- Flexbox
